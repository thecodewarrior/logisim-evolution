Groupes de câbles

Dans un simple circuit Logisim, la plupart des câbles ne transportent qu'un seul bit. Mais Logisim permet également de créer des groupes de câbles (nappes de câbles) qui transportent plusieurs bits. Le nombre de bits circulant le long de ces nappes est la largeur de bitde ce câble.
