Etape 3: Ajouter du texte

Il n'est pas nécessaire d'ajouter du texte dans votre circuit pour le faire fonctionner, mais si vous voulez présenter votre circuit à quelqu'un (comme un enseignant), alors quelques étiquettes permettront de mieux présenter le rôle des différents éléments de votre circuit.

Sélectionnez l'outil texte (). Vous pouvez cliquer sur un pin d'entrée et commencer à taper pour lui assigner une étiquette. (Il est préférable de cliquer directement sur le pin d'entrée plutôt que de cliquer à l'endroit où vous voulez placer le texte, car de cette manière l'étiquette se déplacera avec le pin). Vous pouvez faire de même avec le pin de sortie. Ou vous pouvez juste cliquer sur n'importe quel ancien emplacement et commencer à taper du texte pour mettre une étiquette n'importe où ailleurs.

Beaucoup de composants acceptent des labels, par exemple si vous cliquez dans une des portes vous pouvez lui donner un label

Il est possible de modifier un label de plusieurs autres manières.

    Avec un double clic sur le composant avec l'outil d'édition().
    Avec un clic sur un label avec l'outil text().
    En modifiant la propriété label dans la fenêtre de propriété.
