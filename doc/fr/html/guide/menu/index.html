Référence des menus

Cette section explique les six menus qui accompagnent chaque fenêtre majeure de Logisim.

Le menu Fichier
Le menu Editer
Le menu Projet
Le menu Simulation
Le menu Fenêtre et Aide
de nombreux éléments de menu se rapporte spécifiquement à un projet actuellement ouvert. Mais quelques fenêtres de Logisim (en particulier la fenêtre Analyse combinatoire et la fenêtre Préférences d'application) ne sont pas associés à des projets. Pour ces fenêtres, les éléments de menu spécifiques au projet seront désactivés.

Suiant: Le menu Fichier.
